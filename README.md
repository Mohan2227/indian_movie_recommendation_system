# Indian Movie Recommendation System

## Overview
This project is a content-based movie recommendation system that suggests similar movies based on user input. It uses Natural Language Processing techniques and is deployed using a Flask web application with a dynamic UI.

## Problem Statement
To build a system that recommends movies based on user preferences, similar to platforms like Netflix.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Flask
- HTML/CSS

## Workflow
1. Data preprocessing and feature selection  
2. Combining genre and description  
3. TF-IDF vectorization  
4. Cosine similarity calculation  
5. Flask app for user interaction  
6. Displaying recommendations in UI  

## Features
- Dynamic web interface  
- Real-time movie recommendations  
- Content-based filtering  
- Interactive UI with animations  

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run application:
python app.py

3. Open in browser:
http://127.0.0.1:5000/

## Example

Input:
3 Idiots

Output:
List of similar movies displayed on UI
